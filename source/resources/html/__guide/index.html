웹접근성 유의사항

1) 문서의 흐름 (focus 이동을 놓치지 않아야함)

2) alt(대체텍스트), label = id 매칭, label 없을때 title 빠뜨리지 않기

3) table caption, scope

4) 버튼 사이즈
   - 추천 44*44 이상 (WCAG2.1 : http://www.kwacc.or.kr/WAI/wcag21/)
   - 최소 20*20 이상 (MACAG 2.0 : http://www.webwatch.or.kr/MA/020201.html?MenuCD=220)

5) 명도대비 (눈에 띄는건 디자이너에게 말해주기)

SCSS구성
                            
                                reset : reset
                                font : webfont 호출
                                variables : 변수 모음 (채널별 구분 고려)
                                default : 공통 설정 및 레이아웃
                                sprite : 자동생성 되는 icon의 sprite css (직접 작성하지 않음)
                                atom : heading, button, icon, form
                                molecule : tab, button group, Tooltip
                                organism : header, footer, layer,
                                template : Layout
